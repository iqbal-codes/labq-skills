# UI Tokens

**Last Updated:** 2026-06-23

## Color Tokens

| Token | Value | Usage |
|-------|-------|-------|
| `--color-primary` | `#2563eb` | Primary actions, links |
| `--color-surface` | `#ffffff` | Card and panel backgrounds |
| `--color-border` | `#e5e7eb` | Borders, dividers |
| `--color-muted` | `#6b7280` | Secondary text, void states |
| `--color-success` | `#16a34a` | Positive status (paid) |
| `--color-warning` | `#d97706` | Attention status (pending) |
| `--color-danger` | `#dc2626` | Negative status (overdue, errors) |
| `--color-overlay` | `rgba(0,0,0,0.5)` | Modal backdrop |

## Spacing Tokens

| Token | Value | Usage |
|-------|-------|-------|
| `--spacing-xs` | `0.25rem` | Tight inline gaps |
| `--spacing-sm` | `0.5rem` | Field-to-label spacing |
| `--spacing-md` | `1rem` | Default component padding |
| `--spacing-lg` | `1.5rem` | Section-to-section spacing in settings pages |

## Radius Tokens

| Token | Value | Usage |
|-------|-------|-------|
| `--radius-sm` | `0.25rem` | Input fields |
| `--radius-md` | `0.5rem` | Buttons, cards |
| `--radius-lg` | `0.75rem` | Modals |
| `--radius-full` | `9999px` | Badges, pills |

## Shadow Tokens

| Token | Value | Usage |
|-------|-------|-------|
| `--shadow-sm` | `0 1px 2px rgba(0,0,0,0.05)` | Cards at rest |
| `--shadow-lg` | `0 10px 15px rgba(0,0,0,0.1)` | Toasts, dropdowns |

## Badge Status Colors

| Status | Token | Hex |
|--------|-------|-----|
| paid | `--color-success` | `#16a34a` |
| pending | `--color-warning` | `#d97706` |
| overdue | `--color-danger` | `#dc2626` |
| void | `--color-muted` | `#6b7280` |
